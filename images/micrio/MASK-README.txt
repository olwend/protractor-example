micrio mask creation guide

1) print screen full interactive screen (1920x1080) in desired zoomed-in view , open in PS
2) create a new empty layer (mask), keep both layers visible
3) draw rectangles over required highlight, fill with paintbucket in black
4) crop new mask layer to micrio container borders
5) hide other layers except new mask layer
6) apply gaussian blur (radius 4.0)
6) export as PNG.
